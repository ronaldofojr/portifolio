# Portfolio - Ronaldo Ferreira

Site de portfólio profissional com foco em posicionamento de Project Manager com background técnico.

## Stack

- React + Vite
- Tailwind CSS

## Rodar localmente

1. Instale Node.js (versão 18+)
2. Instale dependências:

npm install

3. Inicie o ambiente de desenvolvimento:

npm run dev

4. Build de produção:

npm run build

## Estrutura

- src/components: componentes reutilizáveis de UI e seções
- src/data/portfolioData.js: conteúdo centralizado do portfólio
- src/App.jsx: composição das seções
